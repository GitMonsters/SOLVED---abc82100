## 🏗️ Architecture

The system employs an **8-limb compound-loop design** — a central CORE intelligence node connected to 8 specialized subsystems: Perception, Reasoning, Memory, Planning, Execution, Evaluation, Learning, and Communication. Data flows continuously around the compound loop, allowing each limb to contribute to and benefit from every cycle.

At the heart of the reasoning engine sits a **64-expert Mixture-of-Experts (MoE) grid** — an 8×8 array of specialized micro-models, each trained on a distinct class of pattern transformation. The gating network dynamically routes problems to the most relevant experts, enabling fine-grained specialization without sacrificing generalization.

## 🤔 What is ARC-AGI?

The **Abstraction and Reasoning Corpus (ARC-AGI)** is a benchmark designed by François Chollet to measure genuine machine intelligence. Unlike traditional benchmarks that reward pattern memorization, ARC-AGI presents novel visual grid puzzles that require core knowledge priors (objectness, geometry, counting, symmetry) to solve — the same priors human children acquire effortlessly. Solving ARC tasks is considered a strong indicator of true generalization ability, making it one of the most challenging and meaningful benchmarks in modern AI research.
